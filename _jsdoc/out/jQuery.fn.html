<static> onoPagerSwipe

Handles swipe events on mobile phones and desktops. The code is based on the QuickGestures jQuery plugin of Anders Zakrisson.
Parameters:
Name Type Description
arg_config Object | Null Swipe config.
Properties
Name Type Description
dragLeft Function Handles drag left event.
dragRight Function Handles drag right event.
dragUp Function Handles drag up event.
dragDown Function Handles drag down event.
threshold Number Number of pixels that have to be dragged before something is considered a swipe.
platform String Determines on what platforms the user is able to page by swiping. 'touch' activates swiping only on touch devices. 'all' will activates swiping on touch devices and desktop. Swiping on the desktop is done with mouse gestures. Default value is 'touch'.
