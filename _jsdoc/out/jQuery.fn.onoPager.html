Class: onoPager

jQuery.fn. onoPager

new onoPager

Creates a widget that enables the user to navigate through a list of elements (like an <ul>). It provides different types of paging navigation. These are called animation objects and reside in the onoPager.animation namespace.
The onoPager library consist of many types of animations like "slides", "linear" and "linearScroller". The onoPager is built in such a way that it's easy to add your own animation objects without having to rewrite the basic paging functionality.
Check out some configuration examples.

arg_config Object Configuration object.
animationConfig Object Optional extra configuration object for the animation object.
arg_config.cssClass String Ads CSS class to the root of the pager.
arg_config.pagePerItem Boolean If true, each page will be as long as one list item. If false, each page will be as long as the list container, which may span multiple list items.
arg_config.lockDuringTransition Boolean Disable paging controls during a transition when true. Default is false.
arg_config.doesLoop Boolean If true, the pager scrolls back to the first item after the last item.

arg_config.listContainer.height String Height of list container, like '200px'.
arg_config.listContainer.maxHeight String Maximum height of list container, like '200px'. The maximum height will also be applied to all list Items.
arg_config.listContainer.adjustHeightToListItem.active Boolean If you also set pagePerItem to true, the height of the list container will adjust to the height of the visible list item. Default value is true.
arg_config.listContainer.adjustHeightToListItem.animate Boolean If true, the container will animatie to its new height. Default value is true.
arg_config.ListItems.width String Width of list items, like '200px'.
arg_config.ListItems.height String Height of list items, like '200px'.
arg_config.activeIndex Number Sets initial visible page. By default the pager starts at index 0.
arg_config.autoPage.active Boolean Activates auto pager.
arg_config.autoPage.interval Number The interval between autopaging. Time value is set in milliseconds.
arg_config.autoPage.active Boolean Activates auto pager.
arg_config.autoPage.autoPageAnimationType String The type of animation that will indicate the time the time between transitions.
arg_config.autoPage.extraConfig Object A configuration object for the autopage indicator.
arg_config.autoPage.pauseOnHover Boolean Stops the pager when the mouse cursor hovers above the pager.
arg_config.labels.next String text for the 'next'-button.
arg_config.labels.previous String Text for the 'previous'-button.
arg_config.status.active Boolean Activates the status box.
arg_config.status.prependText String Text that appears before the page index number.
arg_config.status.seperationText String Text that appears between the page index number and the total pages number.
arg_config.status.appendText String Text that appears after the total pages number.
arg_config.scroller.active Boolean Activates a Javascript scrollbar. Default is true.
arg_config.pixelMove Number The amount of pixels the pager scrolls after each frame.
arg_config.pageByNumber.active Boolean Activates the bar with all pages, defined by number. Default is true.
arg_config.pageByNumber.enableClick Boolean Disables paging behaviour onclick. The makes the 'Page by number'-box essentially a status box rather than a navigation control. Default is true.
arg_config.pageByNumber.labels Array Replaces the default content of the page-by-number links with the string values in the array.
arg_config.pageByNumber.hideThreshold Number Defines how many links may appear in the pageByNumber box. If you set the value to 2, you'll see 2 links on the left and 2 on the right of the active item. A negative value cancels this functionality. Default value is -1.
arg_config.pageByNumber.links Array Overrides the default click behavior of the page-by-number links. If you supply an array of URL's, they will behave like standard links that point to an URL.
arg_config.pageByNumber.enableHover Boolean Initiates a page when the mouse is hovered over a page by number item.
arg_config.pageByArrowKey.active Boolean Enables paging by using the keyboard arrow keys. Default is false.
arg_config.pageByArrowKey.preventDefault Boolean Disables or activates the default behaviour of the arrow key. If set to true, the user won't be able to scroll the page or a textarea with the arrow keys. Default is false for that reason.
arg_config.swipeTriggersPage Boolean Activates page navigation by swiping on the screen. Default is true.
arg_config.swipePlatforms String Determines on what platforms the user is able to page by swiping. 'touch' activates swiping only on touch devices. 'all' will activates swiping on touch devices and desktop. Swiping on the desktop is done with mouse gestures. Default value is 'touch'.
arg_config.animationType String Determines which animation object will be used. The following animation types are available by default: 'linear', 'linearScroller' and 'slides'. Custom animation objects can be created and used after the plugin is loaded. Default value is 'linear'.
arg_config.animationEasing String Determines the easing type to be used by the animation object. Default value is 'linear'. If 'easeOutCubic' is available, it will use that as default value.
arg_config.orientation String Determines on what axis the animation moves. Possible values are 'horizontal' and 'vertical' though it's possible to use other values as long as the animation object supports that value. Default value is 'horizontal'.
arg_config.animationSpeed Number Determines the speed at which the animations take place.

Example
// Simple example:
// More examples on: http://www.thebrightlines.com/onopager/website/
jQuery('#list1').onoPager({
  cssClass: 'onoPager_greyscale',
  pagePerItem: true,
  doesLoop: false,
  listContainer: {
    width: '280px',
    height: ''
  },
  listItems: {
    width: '260px',
    height: '80px'
  },
  status: {
    active: false
  },
  pageByArrowKey: {
    active: true,
    preventDefault: false
  },
  pageByNumber: {
    hideThreshold: 1
  },
  swipeTriggersPage: true,
  animationType: 'linear',
  animationEasing: 'easeOutCubic',
  orientation: 'horizontal',
  animationSpeed: 1000
});
